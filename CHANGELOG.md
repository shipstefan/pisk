# Changelog

All notable changes to `pisk` are documented here.

## [0.1.0] - 2026-07-01

Initial release — 10 skills across three tiers.

### Foundation
- `pisk:launch` — Scaffold the SaaS stack (Next.js, Clerk, Neon, Stripe, Resend, PostHog)

### Context
- `pisk:growth` — Business, team, goals, acquisition, OKRs → `docs/context/`
- `pisk:funnel` — Ideal customer profile and offer → `docs/context/icp.md`, `offer.md`
- `pisk:content` — Story, themes, voice, cadence → `docs/context/content/`
- `pisk:analytics` — Consent, behavior tooling, ad signals, attribution → `docs/context/analytics.md`

### Operations
- `pisk:post` — Draft daily content from the content context
- `pisk:reconcile` — Daily whole-funnel reconciliation report
- `pisk:meta` — Meta Ads reconciliation and optimization loop
- `pisk:linkedin` — LinkedIn Ads reconciliation and optimization loop
- `pisk:ab` — PostHog A/B test setup and review
